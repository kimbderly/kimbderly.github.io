Navigating
with the A* Algorithm
One Route Solution is an interactive transport application focusing on the area of Punggol. It implements Dijkstra and A*star algorithms to determine the shortest and fastest route. Users are able to view the HDB, bus stops and LRTs in the area and navigate by bus, the shortest distance or the fastest route.

MY ROLE

  • Collection of Data Nodes
  • Fastest & Shortest Routes
  • Computing the A* Algorithm

TOOLS

  • LTA Datamall API, Overpass API
  • Python, heapq
