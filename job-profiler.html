In collaboration with 4 others, Job Profiler serves as a convenient platform for users to search for jobs, view company reviews and various analyses on jobs and companies. Reviews undergo sentiment analysis powered by Google Cloud Natural Language API. This app was built in both Java and C++ to practice OOP.

MY ROLE

  • Applying OOP Concepts
  • Web Scraping for Jobs & Reviews
  • Analysis of Job Descriptions

TOOLS

  • Java (Jsoup)
  • C++ (Chilkat, libxml++, nlohmann-json)
